## Find last non-zero digit in n!

### Description

For a given integer n, find the last non-zero digit in the factorial of n.

n! = n * (n-1) * (n-2) * .... * 3 * 2 * 1

### Input

The first line should be the value of n.

### Output

In the first line, print the last non-zero digit of n!.

### Sample Input

10

### Sample Output

8
